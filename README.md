# Automated Clonogenic Assay Analysis 
Code accompanying the manuscript: 'Quantifying measurement bias in clonogenic survival assays using automated colony detection and synthetic benchmarking'.

# Overview
This repository contains the code, experimental assay images, annotation masks, and publication figures used in the manuscript.

The project investigates uncertainty in automated clonogenic assay analysis using a deterministic dual-mode classical image-processing pipeline.

# Repository Contents
project_code.ipynb

Five experimentally acquired clonogenic assay images used for validation

Manual ImageJ annotations corresponding to the experimental assay images.

Figures presented in the manuscript.

# Requirements
Install required packages using:
pip install -r requirements.txt

Open:'project.code.ipynb' and execute all cells sequentially.

# Data availability
Experimental images and corresponding annotation masks are provided for validation and reproducibility purposes.

Synthetic datasets are generated procedurally within the notebook.
